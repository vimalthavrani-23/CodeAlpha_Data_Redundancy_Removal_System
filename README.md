# CodeAlpha_Data_Redundancy_Removal_System
Data Redundancy Removal System using Python and Excel to detect and prevent duplicate records.
# Data Redundancy Removal System

## Description
This project detects and prevents duplicate data before storing it in an Excel database.

## Features
- Add new records
- Detect duplicate ID
- Detect duplicate Email
- Store only unique records
- View all stored records

## Technology Used
- Python
- openpyxl
- Excel (.xlsx)

## How to Run
1. Install Python
2. Install openpyxl:
   pip install openpyxl
3. Run:
   python main.py
